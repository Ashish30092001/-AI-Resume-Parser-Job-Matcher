# -AI-Resume-Parser-Job-Matcher
🚀 An end-to-end AI system that: ✅ Parses any PDF resume using PyMuPDF ✅ Extracts skills &amp; experience using Groq LLaMA3 AI ✅ Scrapes job listings from Naukri &amp; LinkedIn ✅ Matches jobs using TF-IDF Cosine Similarity ✅ Returns ranked job list with % fit score  Built by Ashish Singh | Data Scientist | Prompt Engineer
